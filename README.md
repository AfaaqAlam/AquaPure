# 💧 AquaPure - Water Delivery Landing Page

AquaPure is a modern and responsive water delivery landing page built using **HTML5** and **CSS3**. The website features a clean design, smooth hover effects, and a fully responsive layout for desktop, tablet, and mobile devices.

## 🚀 Features

- Responsive design
- Modern UI
- Hero section
- Services section
- Why Choose Us section
- Statistics section
- Call-to-Action section
- Contact section
- Footer
- Smooth hover effects
- Font Awesome icons
- Google Fonts (Poppins)

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## 📱 Responsive

The website is optimized for:
- Desktop 💻
- Tablet 📱
- Mobile 📱
